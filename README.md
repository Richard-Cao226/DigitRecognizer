# DigitRecognizer
Uses Linear SVC (Support Vector Classifier) to predict digits that the user writes into Pygame window.
## How to run
1. Download the repository
2. Run DigitRecognition/test.py
3. Write a digit in the window that appears
4. Close the window
5. See the result
